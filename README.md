# Homelab

This repository contains all of the configuration for my homelab.

## Purpose

The purpose of my homelab is to learn and to have fun. I run this setup to explore Kubernetes in a personal environment, experimenting with new tools, techniques, and architectures that I can't always test in production. By self-hosting applications on my own infrastructure, I gain hands-on experience with the entire lifecycle—from deployment to maintenance, backup strategies, security considerations, and operational reliability. It's a place where I can take ownership of a system end-to-end and learn what it really takes to keep services running.

## Homelab Details
- Homeserver: apollo
- Hardware: Dell OptiPlex 3050 Micro
- Kubernetes: k3s
